Mod My Ride

Mod My Ride is a web and mobile application designed to help car enthusiasts plan and track modifications to their vehicles. Built with Python and Flet, the app provides users with tools to manage different modifications, track costs, and visualize the custom build process for their cars.


Key features include:

  • Responsive Design: Built with mobile responsiveness in mind, the app works seamlessly across devices, allowing users to update or review mods from their phone, tablet, or desktop.
  • Database Management: Mod My Ride uses a SQL database to store user data securely, ensuring that all modification logs and related information are preserved.

Technology Stack:

  • Backend: Python
  • Frontend: Python Flet
  • Database: MySQL, Firebase
  • Version control: GitHub
